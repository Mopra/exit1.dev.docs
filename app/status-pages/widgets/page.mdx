# Widgets

Widgets are visual components that display monitoring data on your status pages.

## Available Widgets

### Timeline Widget
Displays a chronological timeline of status changes for your checks. Shows when checks went down and when they recovered.

### Uptime Widget
Shows the uptime percentage for each check over a configurable time period. Displays a visual bar chart of availability.

### Incidents Widget
Lists recent incidents (downtime events) with timestamps and duration. Provides a history of service disruptions.

### Downtime Widget
Shows total downtime for each check over a selected period. Useful for SLA reporting and compliance tracking.

### Map Widget
Displays a geographic map showing the monitoring regions and their status. Provides a visual overview of global availability.

### Text Widget
A free-form text block that you can use to add custom messages, maintenance notices, or supplementary information to your status page.

## Widget Configuration

Each widget supports configuration options:

| Option | Description |
|---|---|
| **Title** | Custom title for the widget |
| **Time Range** | Period to display (24h, 7d, 30d, 90d) |
| **Checks** | Which checks to include in the widget |
| **Layout** | Widget size and position on the page |

## Adding Widgets

1. Open your status page settings
2. Click **Add Widget**
3. Select the widget type
4. Configure the widget options
5. Position the widget on the page
6. Save changes

## Removing Widgets

Click the remove button on any widget to remove it from the status page. The underlying monitoring data is not affected.
