## Feature Highlights

### Uptime Monitoring
Monitor websites, APIs, and TCP/UDP endpoints from multiple global regions with checks as frequent as every minute.

### SSL Monitoring
Automatic SSL certificate validation with detailed certificate information, refreshed every 6 hours.

### Domain Intelligence
RDAP-powered domain expiration tracking with configurable alert thresholds at 30, 14, 7, and 1 day(s) before expiry.

### Status Pages
Customizable public or private status pages with widgets and branding options.

### Alerting
Multi-channel alerting via email, SMS, webhooks, Slack, and Discord with smart flap suppression.

### REST API
Read-only REST API with API key authentication for programmatic access to your checks, history, and statistics.
